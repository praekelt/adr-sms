adr-sms
=================

|travis|_



A vumi javascript sandbox app (`vumi-jssandbox-toolkit`_) to handle information flows for the Association for Democratic Reforms, India. 

::

    $ npm install
    $ npm test

Take a look at the `docs`_ for more examples and the toolkit's api.
